# Meeting Notes

A meeting notes system for Obsidian: capture attendees, agenda, decisions, and action items.

## Features
- New Meeting modal (title, date, attendees, type, agenda)
- Structured note with frontmatter + Agenda/Notes/Decisions/Action Items sections
- Action items use checkbox syntax with @owner
- Extract Actions command: copies unchecked items to master Actions note
- Meetings sidebar: recent meetings with title, date, attendee count, action count
- Settings: meetings folder, actions note path
